Dados Livres: plataforma livre de dados abertos



A ideia de desenvolver a plataforma (Dados Livres) surgiu em 2018, quando foi identificado a importância de visualizar informações e tecnologias que usam os dados abertos. O Dados Livres é uma plataforma livre que permite a catalogação de fontes de dados abertos e aplicações cívicas por meio de colaboração coletiva. O projeto é mantido pela estudante e pesquisadora Carolina Soares, orientado pelo Prof. Me. Pedro Baesse e originalmente desenvolvido no IFRN - campus Ceará-Mirim.








Existem iniciativas parecidas como o Brasil.io e o Colaboradados. Porém, alguns diferenciais do Dados Livres é a facilidade de cadastro. São os próprios usuários da plataforma que cadastram as fontes de dados abertos e as aplicações cívicas sem que haja necessidade de codificação. O Dados Livres facilita o trabalho de jornalistas, desenvolvedores e cidadãos ao simplificar a descoberta, a colaboração ou uso dos dados abertos.

Outro diferencial significativo da plataforma, é inovar conectando fontes de dados abertos com as aplicações cívicas. Por exemplo, ao entrar em um perfil de uma aplicação cívica o usuário pode saber quais as fontes de dados abertos que foram utilizadas e vice-versa.

Para incentivar também na construção da plataforma o projeto é software livre com o seu código fonte disponível no GitLab e licenciado sob Licença GNU GPLv3. A página de como contribuir detalha algumas possíveis formas de colaborar com a plataforma.

Concluindo, é esperado que o Dados Livres possa criar um ambiente que facilite a participação da população no controle social e difusão de dados abertos para os mais diversos fins.
