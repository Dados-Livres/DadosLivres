Ficamos felizes por se interessar em contribuir com o projeto! :) Dados Livres, é desenvolvido pensando em ser um ambiente de compartilhamento e principalmente de colaboração de seus usuários. Confira abaixo algumas possíveis maneiras de realizar contribuições:



Sugira melhorias ou ideias

Você pode ajudar contribuindo com qualquer melhoria, incluindo melhora nos textos e links do site ou sugerindo novas ideias para o projeto. Para isso, entre para o grupo Dados Livres no Telegram ou preencha o formulurário na página de contato.


Cadastre Fontes e Aplicações

Se inscreva-se na Plataforma Dados Livres e colabore registrando fontes e aplicações abertas de conhecimento e utilização para todos os públicos. Veja abaixo imagens que ilustram o registro de uma fonte e de uma aplicação por preenchimento de formulário:




Contribuia no desenvolvimento de software

A Plataforma Dados Livres é Software Livre licenciado sob Licença GNU General Public License Version 3 e seu código fonte pode ser acessado na plataforma do GitLab. Confira mais detalhes de como contribuir aqui e para mais detalhes sobre o projeto e como instalar leia o arquivo Readme.me. O sistema usa a linguagem de programação Python e o pequeno framework web Flask.


Confira abaixo algumas possíveis maneiras para colaborar no desenvolvido de software:


Aplique conhecimentos de Design

Aumente as interações e melhore a experiência dos usuários com a interface. Ajude a aplicar os fundamentos de design UX e UI. Contribua para que Dados Livres tenha um Design único de cores e fontes chamativas, além de uma logo mais relacionada com o objetivo da plataforma.

Para saber mais em como ajudar com o Design clique aqui.


Melhore as traduções

Melhore as traduções do sistema, contribua para que Dados Livres seja acessível em outros idiomas. Ajude a traduzir fontes e aplicações que são registradas pelos próprios usuários da plataforma.

Para saber mais em como ajudar com as traduções clique aqui.


Realize testes e melhorias na segurança

Ajude a aplicar testes de software utilizando os conceitos de TDD, relatando bugs e melhorando a qualidade da nossa aplicação. Reforce a segurança da plataforma encontrando possíveis falhas no sistema.

Para saber mais em como ajudar com testes e segurança clique aqui.


Relate por bugs no sistema

Ajude relatando possíveis bugs no sistema.

Para saber mais em como ajudar relatando bugs clique aqui.


Contribua com a documentação

Ajude melhorando o texto de nossa documentação.

Para saber mais em como ajudar com a documentação clique aqui.


Ajude no desenvolvimento da API

Ajude no desenvolvimento de uma API Web para Dados Livres disponibilizada em código aberto.

Para saber mais em como ajudar com a API clique aqui.
