{{_('Como contribuir?')}}



Ficamos felizes que queira contribuir com o projeto! Dados Livres, é desenvolvido pensando em ser um ambiente de compartilhamento e principalmente de colaboração de seus usuários. Confira abaixo algumas possíveis maneiras de fazer contribuições:



Comunidade

A sua primeira contribuição pode ser entrar e participar da nossa comunidade Dados Livres no Telegram.


Fontes e Aplicações

Precisamos de sua ajuda para tornar Dados Livres uma plataforma diversa de fontes e aplicações abertas de conhecimento e utilização para todos os públicos. Qualquer usuário comum, inscrito na plataforma, pode contribuir registrando uma fonte aberta ou uma aplicação aberta com o simples preenchimento de formulários.


Desenvolvimento de Software

Dados Livres é um Software Livre de código aberto disponível em nosso repositório no GitLab, confira algumas de nossas issues e o arquivo Readme.me para saber como instalar. O sistema usa a linguagem de programação Python e a MicroFramework Flask.


Testes

Ajude a aplicar testes de software utilizando os conceitos de TDD. Relatando bugs e melhorando a qualidade da nossa aplicação.


Design

Aumente as interações e melhore a experiência dos usuários com a interface. Ajude a aplicar os fundamentos de design UX e UI. Contribua para que Dados Livres tenha um Design único de cores e fontes chamativas, além de uma logo mais relacionada com o objetivo da plataforma.


Tradução

Melhore as traduções do sistema, contribua para que Dados Livres seja acessível em outros idiomas. Ajude a traduzir fontes e aplicações que são registradas pelos próprios usuários da plataforma.


Documentação

Ajude melhorando o texto de nossa documentação.


Segurança

Reforce a segurança da plataforma. Encontre possíveis falhas no sistema.


API

Ajude no desenvolvimento de uma API Web para Dados Livres disponibilizada em código aberto.
